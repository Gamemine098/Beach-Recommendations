ทะเลหมู่เกาะสิมิลัน

‌ ‌ ‌ ‌ ‌ ‌ ‌ หมู่เกาะสิมิลัน เป็นหมู่เกาะเล็กๆ ในทะเลอันดามันมี 9 เกาะ เรียงจากเหนือ มา ใต้ คือ เกาะหูยง เกาะปายัง เกาะปาหยันเกาะเมี่ยง เกาะปายู เกาะหัวกะโหลก เกาะสิมิลัน และเกาะบางู หมู่เกาะสิมิลัน ได้รับการยกย่องว่าเป็นหมู่เกาะที่มีความสวยงามทั้งบนบกและใต้น้ำ มีปะการังที่สวยงามหลายชนิด

ที่อยู่ : ตำบลลำแก่น อำเภอท้ายเหมือง จังหวัดพังงา

เปิดให้เข้าชม : 15 ตุลาคม-15 พฤษภาคม ของทุกปี

โทร : 0-7645-3272

ทะเลเกาะเหลาลาดิง จ.กระบี่

ทะเลเกาะเหลาลาดิง

‌ ‌ ‌ ‌ ‌ ‌ ‌ เกาะเหลาลาดิง หรือ เกาะพาราไดซ์ เป็นเกาะที่มีขนาดไม่ใหญ่ มีชายหาดเล็ก ๆ มีลักษณะเป็นเวิ้งอ่าวหลบอยู่ด้านในของภูเขาหินปูนสองฝั่ง ที่นี่จึงเงียบสงบ มีบรรยากาศส่วนตัวสุด ๆ เหมาะสำหรับคนที่อยากมานอนริมทะเล พักผ่อนกับธรรมชาติ

ที่อยู่ : เกาะเหลาลาดิง อำเภออ่าวลึก จังหวัดกระบี่

เปิดให้เข้าชม : ทุกวัน

โทร : 0 7568 1096

ทะเลเกาะทะลุ จ.ประจวบคีรีขันธ์

ทะเลเกาะทะลุ

‌ ‌ ‌ ‌ ‌ ‌ ‌ เกาะทะลุเป็นส่วนหนึ่งของอุทยานแห่งชาติเขาแหลมหญ้า  หมู่เกาะเสม็ด อ.แกลง ระยอง เป็นเกาะที่ห่างจากฝั่งเพียงแค่ 15-20 นาที เท่านั้นโดยเรือสปีทโบ๊ท ดังนั้นการพาลูกไปด้วยเป็นเรื่องง่ายมาก เป็นเกาะที่ชายหาดสวย ขาว น้ำใส และปลาเยอะมาก

ที่อยู่ :  อำเภอแกลง จังหวัดระยอง

เปิดให้เข้าชม : เปิดทุกวัน 08.00-18.00 น.

โทร :  038 655 4201

ทะเลเกาะหมาก จ.ตราด

ทะเลเกาะหมาก

‌ ‌ ‌ ‌ ‌ ‌ ‌ ไปทะเลตราดหากถามว่าเลือกไปเที่ยวเกาะไหนก่อนดี ก็คงแนะนำเกาะช้างเพราะไปง่าย แต่หากรู้สึกติดใจก็ไปต่อเกาะหมากกันเลยค่ะ ซึ่งเกาะหมากจะอยู่ตรงกลางระหว่างเกาะช้างกับเกาะกูด เป็นเกาะที่แม่ไข่ไปซ้ำหลายครั้งมีบาร์ริมทะเลที่เริ่ดมาก และราคาไม่แพง

ที่อยู่ : ตำบลเกาะหมาก ในอำเภอเกาะกูด จังหวัดตราด

เปิดให้เข้าชม : สามารถดูตารางเวลาเรือไปเกาะหมากได้จากบริษัทเรือที่ให้บริการ

โทร : 0 3959 7259

ทะเลเกาะราชาใหญ่ จ.ภูเก็ต

ทะเลเกาะราชาใหญ่

‌ ‌ ‌ ‌ ‌ ‌ ‌ เกาะราชา แบ่งออกเป็น เกาะราชาใหญ่ และเกาะราชาน้อย เกาะราชาใหญ่เป็นเกาะที่มีหาดทรายขาวสะอาด มีหาดทางด้านตะวันตกอยู่ระหว่างหุบเขาเป็นรูปคล้าย เกือกม้าเรียกว่า อ่าวปะตก มีหาดทรายขาวละเอียด น้ำทะเลใสสะอาดลักษณะคล้ายทะเลแถบหมู่เกาะสิมิลัน

ที่อยู่ : ตำบลราไวย์ อำเภอเมืองภูเก็ต จังหวัดภูเก็ต

เปิดให้เข้าชม : สามารถเที่ยวชมได้ตามรอบเรือ บริษัททัวร์นำเที่ยว

โทร : -

ทะเลเกาะมันกลาง จ.ระยอง

ทะเลเกาะมันกลาง

‌ ‌ ‌ ‌ ‌ ‌ ‌ เกาะมันกลาง เป็นเกาะขนาดไม่ใหญ่มาก อยู่ถัดออกมาไม่ไกลจากเกาะมันใน จากเกาะมันในถึงเกาะมันกลางใช้เวลาแค่ 10 นาที เป็นเกาะเอกชน สามารถพักบนเกาะได้โดยติดต่อแพ็คเกจทัวร์จากบนฝั่ง บนเกาะมีที่พักอยู่เพียงแห่งเดียว คือ มันตาคีรี ไอส์แลนด์ รีสอร์ท

ที่อยู่ : ตำบล กร่ำ อำเภอแกลง จังหวัดระยอง

เปิดให้เข้าชม : ทุกวัน

โทร : 08 7992 1999, 08 7921 7555, หรือ 0 3865 7232

ทะเลเกาะมุก จ.ตรัง

ทะเลเกาะมุก

‌ ‌ ‌ ‌ ‌ ‌ ‌ เกาะมุกเป็นเกาะที่ใหญ่เป็นอันดับที่ 3 ของท้องทะเลตรัง แบ่งเป็น 2 ฝั่ง คือ ชายฝั่งด้านตะวันตกซึ่งเป็นที่ตั้งของถ้ำมรกต ซึ่งฝั่งนี้จะไม่มีชายหาดและที่พักใดๆ และอีกฝั่ง คือ เกาะมุกฝั่งด้านตะวันออก เป็นที่ตั้งของหมู่บ้านชาวประมง มีรีสอร์ทที่พัก ร้านอาหาร สิ่งอำนวยความสะดวกต่างๆ

ที่อยู่ : เกาะมุก ตำบลเกาะลิบง อำเภอกันตัง จังหวัดตรัง

ทะเลเกาะพีพีดอน จ.กระบี่

ทะเลเกาะพีพีดอน

‌ ‌ ‌ ‌ ‌ ‌ ‌ เกาะพีพีดอน เป็นส่วนหนึ่งของหมู่เกาะพีพี เป็นเกาะที่มีชื่อเสียงดิดอันดับโลก เป็นเกาะใหญ่ที่เป็นศูนย์กลางของเกาะพีพี ทั้งที่พัก ท่าเรือ ร้านอาหาร สิ่งอำนวยความสะดวก ทุกอย่างอยู่ที่เกาะนี้ทั้งหมด

ที่อยู่ : อ่าวนาง อำเภอเมืองกระบี่ จังหวัดกระบี่

ทะเลเกาะนางยวน จ.สุราษฎร์ธานี

ทะเลเกาะนางยวน

‌ ‌ ‌ ‌ ‌ ‌ ‌ เกาะเล็กๆที่อยู่ทางทิศเหนือของเกาะเต่าเป็นเกาะสวรรค์แห่งการพักผ่อนที่ได้รับการยกย่องให้เป็น 1 ใน 10 ของโลก เกาะนางยวน ประกอบด้วยเกาะเล็กๆ 3 เกาะ ถูกเชื่อมเข้าหากันด้วยสันทรายสีขาว ตัดกับน้ำทะเลสีฟ้าใส จนกลายเป็นอ่าวส่วนตัวถึง 3 อ่าว ซึ่งนับเป็นเอกลักษณ์อันโดดเด่น ของเกาะนางยวน

ที่อยู่ : อำเภอเกาะพงัน จังหวัดสุราษฎร์ธานี
